外部からの実験環境へのアクセス
---------------------------------------------------------------------

この実験では， IED の GPU マシンを使います．

IED で実験する分には指示どおり動かせば問題ないはずです．
一方，家などの外部から実験を行うためには，大学側のネットワークにホストをつなぐ必要があります．
これには，大学側と Virtual Private Network (VPN) を形成する必要があります．
詳細は下記を参照してください．

https://www.cc.uec.ac.jp/ug/ja/remote/vpn/index.html

VPN は，外部にある計算機を，仮想ネットワークを介して大学のネットワークにつなぐための仕組みです．
この処理を行うことで，外部の計算機が uec.ac.jp 傘下のネットワークに接続され学内でのマシンとして扱われるようになります．（学内の利用ポリシーに則って使用してください）
